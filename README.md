# Transformer
Трансформер позволяет трансформировать строки XML, JSON, CSV и TOON в любой комбинации

XML -> JSON | XML -> TOON | XML -> CSV 

JSON -> XML | JSON -> TOON | JSON -> CSV

TOON -> JSON | TOON -> XML | TOON -> CSV

CSV -> JSON | CSV -> XML | CSV -> TOON
